# Clicker System (Web + Android)

## Overview
This project is a full clicker system consisting of a web-based backend and an Android mobile application. It allows real-time response collection and visualization for classroom interaction.

---

## System Components

### Web Application (Java Servlets)
- Handles question control and response processing
- Stores and retrieves data from MySQL
- Displays aggregated results

### Android Application
- Allows users to submit responses via mobile interface
- Connects to backend server endpoints

---

## Tools & Environment
- IDE: Visual Studio Code
- Java (Servlets)
- Android Studio (Java, XML)
- Apache Tomcat
- MySQL
- JDBC
- Gradle

---

## System Flow
1. Instructor starts question via web app  
2. Students submit responses via Android app  
3. Responses stored in database  
4. Results displayed in real-time  

---

## Project Structure
- `web-app/` — Java Servlet backend  
- `android-app/` — Android client application  
- `database/` — SQL database file
